# 概要
- フラット族のためのアイテクスチャです
- {SourceAvatar}向けに作ったものです。全フラット族で使用可能です

# インストール方法
- pandraboxの「フラット族コンタクトレンズシステム」をインポートしてください
　- https://pandrabox.booth.pm/items/7946706
- {製品名}.unitypackageをインポートしてください
- Unity上部に「Pan」メニューが追加されるので、そこから「ContactLens/カタログ」を開いてください
- 気に入ったものを選ぶとProjectビューでそれが選択されます。プレハブをアバターに入れると目が変更されます

# アンインストール方法
- プレハブを外すと元に戻ります
- 何かトラブルがあったらPan/ContactLens/RemoveAllを使うとより確実に元に戻ります

# カスタマイズ方法
- ContactLens同梱機能により次の編集が可能です
　- 瞳孔設定…瞳孔を使うかどうか設定できます
　- 瞳孔の色…瞳孔を使う場合表示されます。色を変更できます
　- 瞳孔の透明度…瞳孔を使う場合表示されます。透明度を変更できます。透明度を下げたときのFlat3, If, ヘオン, キューフの瞳孔はテクスチャ表現になるので注意してください
　- 目のスケール…レンズテクスチャの大きさがあっていない時、微調整できます
　- 色相シフト…レンズテクスチャの色を変更できます
- 一部の設定は、変更後「更新」ボタンを押す必要があります

# ライセンス
- 専用アバター並びに専用アバター以外への使用を許可します
- 再配布・流用はご相談下さい
- 個人利用の範囲でレンズテクスチャの改変を許可します
- ContactLens機能による色相・サイズ調整、瞳孔の色等調整を許可します

# 困った時は
- 頭が消えた/色が戻らなくなった/マテリアルエラーになった
　- Pan/ContactLens/RemoveAllを使って下さい
- カスタマイズしても変わらない
　- 更新ボタンを押して下さい
- 変な場所の色が変わった
　- 適用先アバターの設定が使っているアバターと一致していることを確認してください
